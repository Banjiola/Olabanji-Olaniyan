Determination of Air Quality Using Gaussian Process Regression and Other Machine Learning Models

Built a machine learning pipeline to predict benzene concentrations using Gaussian Process Regression (GPR) and other models. After cleaning skewed data caused by -200 null encodings, I extracted temporal features and compared four models. GPR achieved the best performance (RMSE: 2.33, R²: 0.883) and provided uncertainty estimates for more informed environmental decisions. Tools: Python, scikit-learn, pandas, seaborn Models: Linear Regression, SVM, Random Forest, GPR
